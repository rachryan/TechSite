# ConnectFlow - Technology Integration Website

A modern, easy-to-navigate website that helps people feel curious and confident about improving how their technology works together. The site shows how apps and tools can connect to simplify work and reduce frustration.

## Overview

ConnectFlow is designed with simple navigation, calm visuals, plain language, and subtle delight. The focus is on clarity, exploration, and low-pressure next steps.

## Features

- **Intuitive Navigation**: Four clear menu items - How It Works, Solutions, Resources, Get Started
- **Responsive Design**: Mobile-first approach that works beautifully on all devices
- **Calm Visual Design**: Soothing color palette with subtle animations
- **Plain Language**: No jargon, easy-to-understand content
- **Multiple User Journeys**: Supports different visitor goals and comfort levels

## Site Structure

- **Homepage** (`index.html`) - Value proposition and overview
- **How It Works** (`how-it-works.html`) - Educational content about integrations
- **Solutions** (`solutions.html`) - Real-world examples and use cases
- **Resources** (`resources.html`) - Guides, tutorials, and FAQ
- **Get Started** (`contact.html`) - Low-pressure contact page

## Getting Started

### View the Website

Simply open `index.html` in a web browser. No build process or server required!

### Technologies Used

- **HTML5**: Semantic markup for accessibility
- **CSS3**: Modern styling with CSS Grid, Flexbox, and CSS Variables
- **Vanilla JavaScript**: Lightweight interactions and animations
- **No dependencies**: Fast loading, easy maintenance

## Design Principles

1. **Clarity** - Visitors always know where they are and what they can do
2. **Calm** - Nothing is jarring, overwhelming, or pushy  
3. **Curiosity** - Content invites exploration without requiring commitment
4. **Confidence** - Reassuring language and clear guidance throughout

## Key Design Elements

### Color Palette
- **Primary Blue**: #5B8DB8 (trust, reliability)
- **Secondary Green**: #8BA888 (growth, harmony)
- **Accent Orange**: #E8A87C (energy, delight)

### Typography
- System font stack for fast loading and familiarity
- Responsive text sizing with `clamp()`
- Clear hierarchy with consistent spacing

### Animations
- Subtle entrance animations using Intersection Observer
- Gentle connection visualization on homepage
- Smooth hover effects on interactive elements
